# NoticeBoard 🚀

A modern, high-performance SaaS application for managing board notices with real-time updates and a premium user experience.

## ✨ Features

- **Tailwind CSS v4 Integration**: Leveraging the latest in CSS-first design for a blazing fast and beautiful UI.
- **Dynamic Relative Timing**: Notices display time in an intuitive "YouTube/Facebook" style (e.g., *10 hours ago*, *2 days ago*).
- **Premium Design System**: 
  - Custom Inter-based typography.
  - Interactive micro-animations and hover effects.
  - Advanced skeleton loading states for a seamless feel.
- **Admin Management Suite**:
  - Full CRUD operations for notices.
  - User promotion and management dashboard.
  - Real-time statistics and overview.
- **Responsive Layout**: Pixel-perfect design across all devices (Mobile, Tablet, Desktop).

## 🛠️ Technology Stack

- **Frontend**: Vite + React
- **Styling**: Tailwind CSS v4 (Alpha/Beta support)
- **Icons**: Lucide React
- **API Client**: Axios
- **State Management**: React Hooks (useState/useEffect)
- **Notifications**: Custom Toast System

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   cd Client
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Backend Setup**:
   Ensure your backend server is running on `http://localhost:3000`.

## 📂 Project Structure

```text
/Client
  /src
    /components/ui  - Reusable, Tailwind-powered core components
    /pages          - High-level application views (Dashboard, Auth, Admin)
    /utils          - Utility functions (e.g., formatRelativeTime)
    /assets         - Global assets and static files
```
